Optimization

Gradient Descent 🔗

Core optimization method that updates parameters by stepping in the direction of negative gradient.

Applications: Neural Networks, Regression

Lagrangian Multipliers 🔗

Introduces auxiliary variables to solve optimization problems with constraints.

Applications: SVM, Dual Problems

Reinforcement Learning

Q-Learning 🔗

Model-free learning of optimal action-selection policies using value updates based on experience.

Applications: Games, Robotics

Policy Gradient Methods 🔗

Optimizes policy functions directly by estimating gradients from sampled rewards.

Applications: Game AI, Control Systems

Proximal Policy Optimization (PPO) 🔗

Stabilizes policy updates using clipping techniques and trust regions during training.

Applications: Robotics, Simulations

Loss Function

Cross Entropy Loss 🔗

Compares predicted probabilities with true labels, penalizing inaccurate classifications.

Applications: Classification, Language Models

Mean Squared Error (MSE) 🔗

Quantifies average squared difference between predictions and true values.

Applications: Regression, Reconstruction

Math

KL Divergence 🔗

A measure of how one probability distribution diverges from a second, expected distribution.

Applications: GANs, Variational Inference

AI

Autoencoder 🔗

Neural network that learns to encode input data into a compressed representation and reconstruct it back.

Applications: Dimensionality Reduction, Anomaly Detection

Softmax Function 🔗

Converts raw output scores into a probability distribution over possible classes.

Applications: Classification, Reinforcement Learning
